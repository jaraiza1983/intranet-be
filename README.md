# Intranet BE (API)

Clonar proyecto
```
git clone https://github.com/jaraiza1983/intranet-be.git
```

Instalar paquetes
```
yarn install
```

Clonar archivo ```.env-template``` y renombrarlo a ```.env``` 

Cambiar las variables de entorno en ```.env```

Leventar la base de datos
```
docker-compose up -d
```

Levantar el programa 
```
yarn start:dev
```
